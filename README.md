# Salaires de la LPHF

Cette application comprend 1 page pour analyser les salaires de base des joueuses de la **Ligue professionnelle de hockey féminin** (LPHF).
* **Sommaire** : sommaire à haut niveau des salaires de base de la LPHF. 
* **Caractéristiques** : répartition des salaires de base de la LPHF selon différentes caractéristiques des joueuses. 
* **Performances** : répartition des salaires de base de la LPHF selon les performances des joueuses. 

Sources des données : PWHLPA ([référence](https://www.pwhlpa.com/salary-guide)) et API HockeyTech ([référence](https://github.com/IsabelleLefebvre97/PWHL-Data-Reference))  
Récupération et traitement des données : repo GitHub [PWHL-Dashboards](https://github.com/Yu270/PWHL-Dashboards/tree/main/data)
