# Salaires de la LPHF

Cette application comprend x pages pour analyser les salaires de base des joueuses de la **Ligue professionnelle de hockey féminin** (LPHF).

Sources des données : PWHLPA ([référence](https://www.pwhlpa.com/salary-guide)) et API HockeyTech ([référence](https://github.com/IsabelleLefebvre97/PWHL-Data-Reference))  
Récupération et traitement des données : repo GitHub [PWHL-Dashboards](https://github.com/Yu270/PWHL-Dashboards/tree/main/data)
